ID?		 Identyfikatory
Jaki jest cel
stosowania ID?		 Celem stosowania atrybutu id w HTML jest nadanie unikalnego identyfikatora konkretnemu elementowi na stronie
Jakie jest
znaczenie znaku #		 służy do definiowania i odwoływania się do selektora ID.
Jak następuje
wywołanie ID
(wraz z
wtłumaczeniem)?		 Wywołanie ID w CSS z poziomu HTML polega na przypisaniu unikalnego identyfikatora do elementu HTML za pomocą atrybutu id, a następnie odwołaniu się do niego w pliku CSS lub wewnątrz znacznika < style > przy użyciu znaku hasha ( # ).
Uwagi na temat
stosowania ID
(trzy) konieczne
wyliczenia.
  1. Każdy id musi być unikalny w obrębie całego dokumentu HTML. Nie wolno używać tego samego id dla więcej niż jednego elementu.
  2. Wartość id nie może zawierać spacji. Zamiast nich używaj myślników (-) lub podkreślników (_), np. id="main-header".
  3. ID powinno zaczynać się od litery, a nie od cyfry czy znaku specjalnego.
